# Nedz

A project to help autistic patients express their needs

## Getting Started

# The tools used in the project
    1- Firebase ML Kit
    2- Firebase Image Labeling
    3- Firebase Cloud Firestore
    4- Firebase Auth
    5- Voice Recognition
    6- SMS
    7- Local Storage
